## devEops
devEops通过封装Ansible api来处理实际业务运行可能遇到的安装部署运维工作

## Getting started
devEops是一个django项目，依赖于

* gradle
* Python2.7+
* Ansible2.2+
* HTTP文件服务器环境
* 基础YUM源
